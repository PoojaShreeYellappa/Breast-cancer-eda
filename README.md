# Breast Cancer EDA
Exploratory data analysis on the Wisconsin Breast Cancer dataset (569 records).

**Tools:** Python, pandas, matplotlib, seaborn

**Key Findings:**
- Dataset contains 357 benign and 212 malignant cases (63/37 split)
- Malignant tumours have significantly larger mean radius than benign tumours
- Area and perimeter are highly correlated with radius (r > 0.9)
